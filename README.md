# Fixro - AI Bug Fixer

##  About Fixro

Fixro is an AI-powered bug fixing tool that helps developers detect, understand, and fix errors in their code quickly. It is designed to make debugging easier, faster, and more efficient — especially for beginners.

## Features

* Detects bugs in user code
* Suggests fixes using AI
* Fast and simple interface
* Supports multiple file inputs (future)
* Planned web-based UI
  
## Tech Stack

* Python (Core Logic)
* OpenAI API (AI Suggestions)
* JavaScript (Future Frontend)
* HTML/CSS (UI - Planned)

## Project Structure

Fixro/
│── main.py          # Entry point
│── analyzer.py      # Code analysis logic
│── fixer.py         # Suggest fixes
│── utils/           # Helper functions
│── README.md        # Project documentation

## How It Works

1. User inputs code
2. Fixro analyzes the code
3. Detects possible bugs
4. Suggests fixes using AI
5. Returns improved code
